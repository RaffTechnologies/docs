## What is Raff Cloud Platform?

Raff Cloud Platform provides on-demand virtual machines with flexible compute, storage, and networking resources. Deploy VMs in seconds, scale on demand, and pay only for what you use with hourly billing.

## Base URL

All API requests are made to:

```
https://api.rafftechnologies.com
```
