## What is Raff?

Raff is a developer-friendly cloud for running virtual machines, block storage, object storage, managed Kubernetes, and VPC networking — all behind a single REST API and a clean dashboard. Deploy in seconds and pay only for what you provision.

## Base URL

```
https://api.rafftechnologies.com
```

## Getting Started

<Steps>
  <Step title="Get your API key">
    Open the [Raff Dashboard](https://app.rafftechnologies.com), click **API Keys** in the sidebar, then **Create API Key**. Copy the key immediately — it's only shown once.
  </Step>
  <Step title="Make your first API call">
    Follow the [Quickstart guide](/quickstart) to list your VMs and create your first virtual machine.
  </Step>
  <Step title="Explore the API">
    Browse the [API Reference](/api-reference/introduction) for all available endpoints with interactive examples.
  </Step>
</Steps>
