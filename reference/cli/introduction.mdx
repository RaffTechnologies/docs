description: "Control your Raff Cloud infrastructure from the command line."
---

<Info>The Raff CLI is coming soon. In the meantime, you can manage your resources using the [REST API](/api-reference/introduction).</Info>

The Raff CLI will let you create, manage, and monitor cloud resources directly from your terminal. It will support all operations available through the API, including:

- Creating and managing virtual machines
- Managing volumes, snapshots, and backups
- Configuring networking and security groups
- Viewing usage and billing information

Stay tuned for updates.
