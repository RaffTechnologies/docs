title: "Raff Terraform Provider"
description: "Provision and manage Raff Cloud resources with infrastructure as code."
---

<Info>The Raff Terraform Provider is coming soon. In the meantime, you can manage your resources using the [REST API](/api-reference/introduction).</Info>

The Raff Terraform Provider will let you define your cloud infrastructure declaratively using HashiCorp Terraform. You'll be able to:

- Provision virtual machines, volumes, and networking
- Manage snapshots and backup schedules
- Version control your infrastructure configuration
- Integrate with existing Terraform workflows

Stay tuned for updates.
