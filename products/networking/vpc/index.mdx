<sub>Updated March 12, 2026</sub>

Virtual Private Cloud (VPC) provides isolated private networking for your Raff Cloud resources. Every VM is automatically placed in a VPC, and you can create shared VPCs to connect multiple VMs on a private network with custom CIDR ranges.
